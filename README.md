# 👨‍💼 Employee Management System

A simple **Python Command Line Interface (CLI)** application to manage employee records. This project allows users to add, update, delete, search, and view employee details using file handling with the `pickle` module.

---

## 📌 Features

- ➕ Add Employee
- ✏️ Update Employee Details
- ❌ Delete Employee
- 🔍 Search Employee
- 👤 View Single Employee
- 📋 View All Employees
- 🚪 Exit Application
- 💾 File Handling using Pickle
- 📂 Modular Programming

---

## 🛠️ Technologies Used

- Python 3
- Functions
- Modules
- File Handling (`pickle`)
- Lists
- Loops
- Match-Case Statement
- Exception Handling

## ▶️ How to Run

1. Clone the repository.

```bash
git clone https://github.com/Jayanthi740/Employee-Management-System.git
```

2. Open the project folder.

```bash
cd Employee-Management-System
```

3. Run the main file.

```bash
python EmpMain.py
```

## 💾 Data Storage

Employee records are stored in a binary file using Python's **pickle** module.

Each employee record contains:

- Employee Number
- Employee Name
- Employee Salary
